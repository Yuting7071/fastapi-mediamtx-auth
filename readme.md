# FastAPI-MediaMTX-Auth

這是一個基於 **FastAPI** 與 **MediaMTX** 的輕量級即時監控方案，支援 **JWT 身份驗證** 與 **WebRTC (WHEP)** 低延遲串流。

## 🚀 功能特點

- **安全驗證**：整合 OAuth2 與 JWT Token 機制，確保串流不被非法存取。
- **低延遲體驗**：採用 WHEP 協議，提供毫秒等級的即時監控畫面。
- **架構清晰**：前端採用原生 JS + HTML，後端採用高效能 FastAPI。

---

## 🛠️ 環境需求

- **Python 3.8+**
- **MediaMTX**：請至 [MediaMTX GitHub](https://github.com/bluenviron/mediamtx/releases) 下載對應作業系統的執行檔。
